# 测试策略文档

项目名称：ModelGate  
版本：v1.0  
第一版范围：无登录、本地存储、MiMo / MiniMax / 火山 Coding Plan Chat 优先，Seedance 暂不接入。

---

## 1. 目标

测试目标：

- 保证 Model Registry 配置正确。
- 保证 Capability Router 推荐正确。
- 保证 Provider Adapter 参数映射正确。
- 保证文件上传安全。
- 保证 Chat Runtime 主链路可用。
- 保证错误码和日志脱敏可用。

---

## 2. 测试分层

```text
unit tests
↓
integration tests
↓
E2E tests
↓
manual smoke tests
```

第一版优先级：

1. 配置校验测试。
2. Capability Router 测试。
3. Adapter mock 测试。
4. API 集成测试。
5. 前端基础 E2E。

---

## 3. 单元测试

### 3.1 Model Registry

测试：

- provider 存在。
- model 引用的 provider 存在。
- paramsSchema 存在。
- capability 合法。
- taskType 合法。
- disabled 模型不会被推荐。
- Seedance 第一版 disabled。

### 3.2 Capability Router

fixture：

- MiMo Chat 模型。
- MiniMax Chat 模型。
- 火山 Coding Plan Chat 模型。
- disabled Seedance 模型。

测试：

- chat 推荐 Chat 模型。
- coding 推荐 code 能力模型。
- image_to_video 不推荐 disabled Seedance。
- provider disabled 时模型不返回。
- hiddenModels 给出原因。

### 3.3 Params Schema

测试：

- required 校验。
- type 校验。
- min / max 校验。
- options 校验。
- providerMapping 校验。
- schema 外字段拒绝。

### 3.4 状态机

测试：

- queued -> running。
- running -> completed。
- completed 不能回到 running。
- cancelled 不能回到 processing。
- Idempotency-Key 重复请求不创建第二个任务。

## 6. 前端 E2E 测试

工具建议：

```text
Playwright
```

场景：

- 打开任务中心。
- 选择 chat 任务。
- 模型列表出现 MiMo / MiniMax / 火山 Coding Plan。
- 选择模型。
- 输入 prompt。
- 运行并展示结果。
- 上传文件并显示状态。
- Provider 错误时展示 requestId。

第一版不要求：

- Seedance 视频生成 E2E。
- 多用户权限 E2E。
- MCP / OpenAI-compatible 外部接口 E2E。

---

## 7. 安全测试

必须覆盖：

- 前端构建产物不包含 API Key。
- `.env` 不进入 git。
- Authorization 日志脱敏。
- api-key 日志脱敏。
- 文件名路径穿越被拒绝。
- SVG / HTML 不 inline 执行。
- 上传文件物理路径不返回前端。

---

## 8. 手动 Smoke Test

每次合并前手动检查：

1. 启动 PostgreSQL 和 Redis。
2. 启动 FastAPI。
3. 启动前端。
4. `GET /health` 正常。
5. `GET /api/providers` 正常。
6. `POST /api/models/recommend` 正常。
7. Chat run mock 正常。
8. request_logs 能查到。
9. 日志无 API Key。

## 9. CI 自动化

工作流文件：`.github/workflows/ci.yml`

触发：

- `push` 到 `main` / `master`。
- `pull_request` 到 `main` / `master`。

后端 job：

- Python 3.11。
- PostgreSQL 16 + Redis 7 service。
- `pip install -e "apps/server[dev]"`。
- `ruff check apps/server/app tests`。
- `python -m compileall apps/server/app`。
- `alembic -c apps/server/alembic.ini upgrade head`。
- `pytest -q`。

前端 job：

- Node.js 22。
- `npm ci`。
- `npm run web:lint`。
- `npm run web:typecheck`。
- `npm run web:build`。
- `npx playwright install --with-deps chromium`。
- `npm run e2e`。
