# ModelGate 待开发文档

更新时间：2026-06-03

本文档汇总 ModelGate 仍待开发的事项，按 P0 / P1 / P2 优先级组织，便于当成行动看板使用。

每条事项标注：

- 状态：`[ ]` 未开始 / `[~]` 进行中 / `[X]` 已完成。
- 来源：对应的 `项目总TODO.md` Phase 编号或外部条件。
- 影响：对应的功能链路或验收要求。

相关文档：

- `前端联通审计.md`：前端控件与后端 API 联通状态。
- `项目总TODO.md`：完整 Phase 任务清单与已完成项。
- `Phase9测试与验收清单.md`：第一版验收范围与命令。

维护规则：

- 任何已联通 / 已验证的审计项不写在本文件，统一进入 `前端联通审计.md`。
- 完成一项时同步在本文件对应位置打勾，并在 `项目总TODO.md` 同步。
- 新增事项先确定优先级（P0 / P1 / P2），再决定是否需要进入 P0 紧急通道。
