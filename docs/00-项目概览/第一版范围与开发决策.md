# 第一版范围与开发决策

项目名称：ModelGate  
版本：v1.0  
决策日期：2026-05-27  
用途：记录第一版正式开发前已经确认的产品和技术边界，作为后续实现、配置和验收的依据。

---

## 1. 第一版定位

第一版定位为面向个人开发者的开源 GitHub 项目。

目标：

- 方便个人开发者 clone 项目后，本地配置自己的 token plan API Key。
- 用统一工作台更方便地调用不同 Provider 的 token plan 模型。
- 第一版优先完成 Chat / Coding 类模型的统一接入。

第一版不做：

- 不做登录系统。
- 不做多用户 SaaS。
- 不做团队权限。
- 不做公开匿名在线服务。
- 不接入火山 Seedance 视频生成。

---

## 2. 登录与用户体系

确认结果：

```text
第一版不需要登录。
```

设计影响：

- 第一版按本地单用户项目设计。
- 不实现 user authentication。
- 不实现 session / token 管理。
- 数据表可预留未来 `user_id`，但第一版不启用。
- API 默认运行在本地或开发者自部署环境。

安全要求：

- 即使无登录，API Key 仍只能放在后端环境变量。
- 不允许前端保存 Provider API Key。
- 不建议第一版直接开放到公网。

---

## 3. Provider 接入顺序

确认顺序：

1. Xiaomi MiMo。
2. MiniMax。
3. 火山 Coding Plan。
4. 火山 Seedance，暂不接入，只保留架构预留。

实现策略：

- 第一版先完成 Chat Runtime。
- Generation Runtime、异步任务和 Worker 结构可以保留，但不把 Seedance 作为第一版必须接入项。
- 火山 Seedance 相关文档保留为后续扩展依据。

## 5. 火山 Seedance 决策

确认结果：

```text
火山 Seedance 第一版先不接入。
```

设计影响：

- `volcengine_seedance_video_schema` 可保留为后续参考，但不进入第一版启用模型列表。
- `text_to_video`、`image_to_video`、`first_last_frame_video` 可作为 taskType 保留。
- Generation Runtime 可保留架构设计，但第一版验收不要求跑通 Seedance。
- `项目总TODO.md` 中 Seedance 接入项应标记为后续版本。

---

## 6. 文件大小限制

确认结果：

第一版先采用文档默认值：

| 类型 | 单文件上限 |
|---|---|
| image | 20 MB |
| video | 500 MB |
| audio | 100 MB |
| document | 100 MB |
| code | 5 MB |

后续优化方向：

- 根据每个模型的上下文长度优化文档解析和上下文截断。
- 根据 Provider 文件输入限制优化上传与转换策略。

---

## 7. 存储方式

确认结果：

```text
第一版先使用本地存储。
```

目录：

```text
storage/uploads
storage/outputs
storage/previews
```

设计影响：

- 第一版实现 `LocalStorageService`。
- `ObjectStorageService` 只预留接口。
- API 不返回物理路径，只返回后端逻辑 URL。

---

## 8. 流式输出

确认结果：

```text
流式输出不是第一版硬性要求。
```

实现策略：

- 如果实现难度可控，可以在第一版支持 SSE。
- 如果影响主链路开发，第一版先支持非流式 Chat。
- API 和 Adapter 设计保留 `stream` 字段和 `stream_chat` 能力。
- 前端 UI 不应依赖流式输出才能完成核心任务。

---

## 9. 第一版验收重点

第一版优先验收：

- 本地无登录运行。
- `.env` 配置 Provider API Key。
- MiMo Chat 可调用。
- MiniMax Chat 可调用。
- 火山 Coding Plan Chat 可调用。
- Model Registry 能管理第一版模型清单。
- Capability Router 能按任务推荐 Chat 类模型。
- 文件上传和基础解析可用。
- 历史记录和请求日志可查询。
- API Key 不出现在前端和日志中。

第一版不验收：

- 火山 Seedance 视频生成。
- 多用户权限。
- 团队空间。
- MCP Server。
- OpenAI-compatible endpoint。
- 自动 fallback。
- 多模型评测。
