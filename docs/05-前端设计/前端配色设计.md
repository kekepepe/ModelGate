# ModelGate 前端配色与视觉风格方案

## 1. 设计方向

ModelGate 的前端不采用深色科技风，也不采用纯白 SaaS 后台风格。整体希望接近一种“低饱和、浅色、纸感、温和科技感”的视觉方向。

参考气质可以理解为 Claude 那类温和、克制、低饱和的产品体验，但不要直接复制 Claude 的颜色。ModelGate 应该更偏向一个“浅色 AI 工作台 / Developer Console”，既有专业感，又不会冷冰冰。

核心关键词：

- Warm Minimal AI Console
- 低饱和浅色工作台
- 纸感卡片
- 柔和边框
- 克制绿色主色
- 温和科技感

## 2. 推荐色彩方案

整体采用：

- 暖灰米白作为页面底色
- 鼠尾草绿作为主按钮和高亮色
- 雾蓝、雾紫、暖橙作为辅助色
- 深棕灰作为文字色
- 柔和米灰作为边框色

```css
:root {
  /* 页面底色 */
  --bg-main: #F6F3EE;
  --bg-soft: #EFEAE2;
  --bg-panel: #FFFFFF;
  --bg-card: #FBFAF7;

  /* 边框 */
  --border-soft: #DDD6CB;
  --border-medium: #CFC6B8;

  /* 文字 */
  --text-main: #2F2A24;
  --text-secondary: #6F675D;
  --text-muted: #9A9083;

  /* 主色 */
  --primary: #7D8F73;
  --primary-hover: #687A5F;

  /* 辅助色 */
  --accent-blue: #8A9BB8;
  --accent-purple: #A395B8;
  --accent-orange: #C99A6B;
  --accent-green: #8FAE8B;

  /* 状态色 */
  --success: #7FA37A;
  --warning: #C89A57;
  --danger: #C9786F;
}
```

## 3. 页面背景

主背景不要用纯白，而是使用暖米灰，增强纸感和柔和感。

```css
body {
  background: #F6F3EE;
  color: #2F2A24;
}
```

也可以增加很轻的背景光感：

```css
body {
  background:
    radial-gradient(circle at top left, rgba(125, 143, 115, 0.14), transparent 32%),
    radial-gradient(circle at top right, rgba(138, 155, 184, 0.12), transparent 28%),
    #F6F3EE;
}
```

要求：背景要有层次，但不能花。整体保持干净、安静、低饱和。

## 4. 侧边栏设计

侧边栏使用比主背景略深一点的暖灰色。

```css
.sidebar {
  background: #EFEAE2;
  border-right: 1px solid #DDD6CB;
}
```

导航选中项不要用高饱和蓝色，使用低饱和背景和绿色左边线。

```css
.nav-item.active {
  background: #E2DED3;
  color: #2F2A24;
  border-left: 3px solid #7D8F73;
}
```

## 5. 卡片 / 面板设计

卡片使用接近纸张的暖白色，不要纯白到刺眼。边框要柔和，阴影要轻。

```css
.card {
  background: #FBFAF7;
  border: 1px solid #DDD6CB;
  border-radius: 18px;
  box-shadow: 0 12px 36px rgba(72, 60, 45, 0.06);
}
```

卡片风格目标：像纸质模块，而不是传统后台的硬质白色组件。

## 6. 按钮设计

主按钮使用鼠尾草绿，表达稳定、自然、克制的 AI 产品感。

```css
.button-primary {
  background: #7D8F73;
  color: #FFFFFF;
  border-radius: 12px;
}
```

Hover 状态：

```css
.button-primary:hover {
  background: #687A5F;
}
```

次按钮使用白色背景和浅边框。

```css
.button-secondary {
  background: #FFFFFF;
  color: #4D463E;
  border: 1px solid #D8D0C4;
  border-radius: 12px;
}
```

## 7. 输入框 / Prompt 框

输入框使用白色或暖白色背景，边框柔和。Focus 状态使用主绿色，但透明度要低。

```css
.input-box {
  background: #FFFFFF;
  border: 1px solid #D8D0C4;
  color: #2F2A24;
  border-radius: 16px;
}
```

```css
.input-box:focus {
  border-color: #7D8F73;
  box-shadow: 0 0 0 3px rgba(125, 143, 115, 0.16);
}
```

## 8. 模型标签颜色

ModelGate 会有不同模型和能力标签，例如 Chat、Code、Image、Video、Vision。标签颜色也要保持低饱和，不要使用传统后台那种鲜艳色块。

```css
.model-chat {
  background: #E7E2D8;
  color: #5F564C;
}

.model-code {
  background: #E4E8DD;
  color: #53664B;
}

.model-image {
  background: #E6E0EA;
  color: #655876;
}

.model-video {
  background: #E7E2D3;
  color: #78613E;
}

.model-vision {
  background: #DDE5EA;
  color: #4F6472;
}
```

## 9. 推荐最终色彩表

| 用途              | 颜色        |
| ----------------- | ----------- |
| 页面背景          | `#F6F3EE` |
| 次级背景 / 侧边栏 | `#EFEAE2` |
| 面板背景          | `#FFFFFF` |
| 卡片背景          | `#FBFAF7` |
| 主文字            | `#2F2A24` |
| 次文字            | `#6F675D` |
| 弱文字            | `#9A9083` |
| 主按钮 / 重点色   | `#7D8F73` |
| 主按钮 hover      | `#687A5F` |
| 辅助蓝            | `#8A9BB8` |
| 辅助紫            | `#A395B8` |
| 辅助橙            | `#C99A6B` |
| 边框              | `#DDD6CB` |

## 10. 设计注意事项

* 不要使用深色背景。
* 不要使用纯白大面积背景。
* 不要使用高饱和蓝紫色作为主色。
* 不要做成传统 SaaS 后台风。
* 页面要温和、克制、舒服，适合长时间使用。
* 所有组件都应保持低饱和、柔和边框、轻阴影。
* 主按钮和关键状态可以使用鼠尾草绿，但不能过亮。
* 模型标签可以有不同颜色，但都必须是雾面、低饱和色。
* 整体要像一个高级、温和、产品化的 AI 工作台。
