## 本地保留、不上 GitHub 的内容

以下目录 / 文件在 `.gitignore` 内，仅保留在本地磁盘，不进入版本控制，也不发布到 GitHub。新协作者从仓库 clone 下来后**没有**这些内容，需要单独获取或忽略。

| 路径 | 用途 |
|---|---|
| `**/前端设计参考图/` | 第三方页面截图（设计参考），版权不属于本项目 |
| `**/tokenplan_api文档/` | 4 个上游厂商的 API 文档，版权通常归厂商 |
| `*-修改建议.md` | 内部评估反馈（如 `minimax3-修改建议.md`），含项目安全风险细节，不宜公开发 |
| `newidea/` | 新版本更新前的 idea 阶段内容，只留在本地，不进入 Git |

具体 ignore 规则见仓库根目录的 [`.gitignore`](../.gitignore)。
